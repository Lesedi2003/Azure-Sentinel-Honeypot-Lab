# Architecture

This folder contains the architecture diagram used throughout the project.

The lab consists of:

- Azure Virtual Machine (Windows Server Honeypot)
- Azure Virtual Network
- Network Security Group
- Azure Monitor Agent
- Log Analytics Workspace
- Microsoft Sentinel
- GeoIP Watchlist
- Microsoft Sentinel Workbook

The diagram illustrates how Windows Security Events travel from the virtual machine into Microsoft Sentinel for investigation and visualization.
