# Azure Sentinel Honeypot Attack Detection and Investigation Lab

## 📖 Overview

This project demonstrates the deployment of an Azure-based Security Operations Center (SOC) lab using Microsoft Sentinel. The lab consists of an internet-exposed Windows virtual machine configured as a honeypot to attract malicious login attempts. Security telemetry is collected, analyzed, and investigated using Microsoft Sentinel, Log Analytics Workspace, and Kusto Query Language (KQL).

The purpose of this project is to develop practical experience in:

- Security monitoring
- Threat detection
- Incident investigation
- Log analysis
- Kusto Query Language (KQL)
- Microsoft Sentinel
- Security Operations Center (SOC) workflows

---

## 🎯 Objectives

The objectives of this lab are to:

- Deploy a Windows honeypot in Microsoft Azure
- Configure Microsoft Sentinel as the SIEM platform
- Collect security telemetry using Log Analytics Workspace
- Monitor unauthorized login attempts
- Investigate security events using KQL
- Create and analyze security incidents
- Develop practical SOC investigation skills
